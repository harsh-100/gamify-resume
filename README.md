# Gamify — Harsh's Dev Quest

Play here 👉 https://game.harshagarwal.dev/

A side-scrolling pixel-art platformer that walks visitors through Harsh Agarwal's journey from a curious B.Tech student in Jaipur to a Full-Stack & Power Platform developer pursuing his Master's in Dortmund, Germany. Each level is a chapter of the journey. Finishing the game unlocks a downloadable resume.

This folder is a **separate app** from the parent Next.js portfolio. It will be lifted out of this repo and deployed independently to `gamify.harshagarwal.dev`.

## Stack

- **Next.js** (App Router or Pages Router — TBD in Phase 1)
- **Phaser.js 3** for the game engine
- **Pixel art** (retro 8/16-bit aesthetic)
- **Vercel** for free hosting on the subdomain

## End-of-game reward

A downloadable resume / CV. Players who finish the journey get the file.

## Journey arc (chapters → levels)

1. **Jaipur — College** (Anand International, B.Tech CS, 2019–2023) — curiosity, first lines of code
2. **Tribal Connect** — Web Dev Intern (Dec 2021 – Feb 2022) — HTML/CSS/PHP/MySQL
3. **IMG Global Infotech** — Node.js Intern (Jun – Aug 2022) — first backend
4. **Ongraph — Trainee** (Oct 2022 – Apr 2023) — JS, React, Node, Express, MongoDB
5. **Ongraph — Software Developer** (Apr 2023 – Jul 2024) — Next.js, Supabase, Playwright, Prisma
6. **Celebal Technologies** — Power Apps & Power Automate Developer (Jul 2024 – Aug 2025)
7. **Dortmund — Master's** (FH Dortmund, 2025 – 2027) — Cloud, Docker, K8s, System Design
8. **Projects Showcase room** — portals into Bc2Bc, Zero to Scalable, Audigram, Diamond E-Commerce, Root Care AI, AI Research Agent
9. **Finale** — resume download + contact CTA
